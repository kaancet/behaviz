# Hover tooltips

!!! note "Stub — to be expanded"

Opt-in interactive tooltips. Most meaningful on **bokeh**; matplotlib uses an annotation-on-
hover engine.

```python
bv.plot_scatter(x, y, hover=True)
bv.plot_scatter(x, y, hover={"label": "trial"})   # custom tooltip label
```
